# ScreenNail (屏幕钉) 📌

一个极简、现代、带有一点赛博朋克风的 Windows 窗口置顶与锁定工具。

相比于传统的置顶软件，ScreenNail 不仅能让窗口置顶，还能**彻底锁死窗口位置（不可拖动）**，并引入了现代化的霓虹边框与半透明“幽灵模式”。

## ✨ 核心特性

- **🔒 绝对锁定**：钉住后，窗口被死死焊在屏幕上，拒绝任何误拖拽。
- **🌈 霓虹圆角边框**：抛弃丑陋的传统图钉图标，使用匹配 Win11 的圆角渐变发光边框。
- **👻 幽灵模式 (Ghost Mode)**：
  - 按住 `Alt` + 滚轮 丝滑调节窗口透明度（带 OSD 屏显提示）。
  - 支持开启“鼠标穿透”，可以透过锁定的窗口点击后面的内容（抄代码/画图神器）。
- **⚡ 极简交互**：快捷键一键切换，同一时间只锁定一个焦点窗口，自动释放旧窗口。

## 🚀 快速使用

1. 在 [Releases](../../releases) 页面下载最新版的 `ScreenNail.exe`。
2. 双击运行（完全绿色免安装）。
3. 激活你想钉住的窗口，按下快捷键 `Ctrl + Shift + P`。
4. 尽情享受！

## ⌨️ 快捷键与操作

| 操作 | 说明 |
| --- | --- |
| `Ctrl + Shift + P` | 钉住当前焦点窗口（如果已钉住，则解锁） |
| `Alt + 鼠标滚轮` | 调节被钉住窗口的透明度 |

## 🛠️ 本地编译构建

本项目使用 C# (WinForms) 与 .NET 10 开发，纯原生调用 Windows API。

```bash
# 运行测试
dotnet run

# 打包为单文件 exe (开箱即用)
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true
```
